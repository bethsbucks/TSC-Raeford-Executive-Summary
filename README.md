# TSC Raeford Executive Summary

Static dashboard files for the Tractor Supply Company Raeford weekly executive summary.

## Live dashboard

Once GitHub Pages finishes deploying, open:

- https://bethsbucks.github.io/TSC-Raeford-Executive-Summary/

## Files

- `index.html` — GitHub Pages entry point
- `TSC Raeford Weekly Dashboard v2.dc.html` — original exported dashboard file
- `support.js` — runtime support script required by the dashboard export
- `.thumbnail` — export thumbnail artifact

## Notes

This repo is a straight publish/test container for the dashboard export so it can be reviewed in GitHub and rendered through GitHub Pages.
